Rules of the Battle

LIFE ARE 100 POINTS AT START OF THE BATTLE.

ATTACK BUTTON => 30 Damage Points on opponent.
· ( If under "spell" does 15 Damage Points )


SPELL BUTTON => 20 Damage Points on opponent.
· ( Spells opponent for 2 more turns. )
· ( Also does some magic )

EAT PIZZA BUTTON => Player recovers 15 life points.
· ( Can ONLY use it IF life is <= 50 )
· ( Takes away spell effect on player )
